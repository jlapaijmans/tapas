---
title:
  - Dependencies of the scripts
---

The scripts consisting this software depend on 

  * A Python 3 installation and Pip, the Python package manager
  * R

Download TAPAS from [this
link](https://github.com/mlell/tapas/releases). Then extract the
downloaded .tar.gz file on your system. 

Then open the README file, which you can find in the TAPAS folder,
or [on the GitHub page](https://github.com/mlell/tapas),
for installation instructions.
